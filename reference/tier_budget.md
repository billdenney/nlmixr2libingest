# Token budget for a complexity tier

Token budget for a complexity tier

## Usage

``` r
tier_budget(tier, budgets = .budgetDefaults())
```

## Arguments

- tier:

  `"low"`/`"medium"`/`"high"`.

- budgets:

  Named tier-\>tokens vector (default via option
  `nlmixr2libingest.tier_budgets`).

## Value

Integer token budget.
